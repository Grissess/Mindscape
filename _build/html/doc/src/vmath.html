vmath – Vector Math¶

This module defines some useful vector and matrix math, implemented generally in numpy for speed.

class vmath.Vector[source]¶

Bases: numpy.ndarray

A vector is an array of double-precision floating points of arbitrary size (though usually between two and four members). The Vector class is actually the numpy.ndarray class, with a few special members, a fixed type, and a special constructor (you may pass elements as arbitrary positional arguments; keyword arguments are not supported).

Vectors can be converted to other vectors of different cardinality by using the To2(), To3(), and To4() methods (or their fast equivalents, described below).

x¶

The x component (item 0).

y¶

The y component (item 1).

z¶

The z component (item 2).

w¶

The w component (item 3).

length()[source]¶

Returns the euclidean length of the vector.

unit()[source]¶

Returns a Vector of unit length.

Warning

This will raise ZeroDivisonError if you pass a zero length vector!

cross(other)[source]¶

Returns the cross product of the provided vectors.

Note

Both vectors must have exactly three elements; you can use To3() or FastTo3() to ensure this.

To2()[source]¶

Returns a duplicate Vector with only two elements.

To3()[source]¶

Returns a duplicate Vector with only three elements.

To4()[source]¶

Returns a duplicate Vector with only four elements.

Note

The returned Vector will have a w (index 3) of 1, if it was not previously present.

FastTo2()[source]¶

Returns a Vector with two elements.

Warning

There is no guarantee that this is not the same Vector object! In fact, this will be the case if the Vector already has the requested length. While this will avoid an uneccessary object construction/ data copy, it means that safety in contexts where the vector is to be mutated cannot be guaranteed. This should only be used when you need to pass the vector as an (immutable) argument to another function that only accepts, e.g., tuples or lists (of the specified size).

Warning

class vmath.Matrix[source]¶

Bases: numpy.matrixlib.defmatrix.matrix

A matrix is a two-dimensional collection of double-precision floating point numbers wherein each dimension has the same cardinality (technically, that means all of the matrices hereby defined are square matrices; for all other cases, you’ll want to use a numpy.ndarray of two dimensions instead.) The class is implemented thinly on top of the numpy.matrix class, and provides only additional operations (no new constructor syntaxes) and new constructors.

classmethod Rotation(angle, axis)[source]¶

Constructs a 3D rotation matrix about the given Vector (which is converted to a unit-length 3D vector) by the specified angle. The domain of the angle is determined by the domain of the underlying numpy.sin and numpy.cos functions (which is radians).

Note

To specify the angle in degrees, use Python’s math.radians function.
