Welcome to Mindscape’s documentation!¶

Mindscape is a game engine designed to be simple and lightweight, but fast and powerful. It was designed for the express purpose of implementing the game of the same name.

The documentation is divided into various sections for respective audiences; feel free to choose one of the following
